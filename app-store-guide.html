Review Information

Notes for App Review (copy-paste)

HydroFlow is a water tracking and hydration reminder app. It does not require an account or login.


HealthKit: The app reads weight, workouts, and sleep data from HealthKit to personalise hydration goals. It writes water intake back to HealthKit. The usage description is shown during onboarding before the permission prompt. HealthKit features are entirely optional — the app functions without granting access.


Location: Location is used only for Smart Reminders (Pro feature) — to trigger hydration nudges when the user arrives at the gym, office, or outdoors. Location permission is requested only when the user enables this feature. The app does not transmit location data externally.


Notifications: Reminders are local push notifications only — no server-side push. The user controls all reminder schedules in Settings.


In-App Purchases: Three subscription tiers are available (Monthly $2.99, Annual $19.99 with 7-day free trial, Lifetime $49.99). Premium features are clearly marked with a lock icon. The app is fully usable without purchasing — free tier includes core water logging, 5 garden plants, basic reminders, and daily stats.


Test note: In the Sandbox environment, StoreKit purchases can be tested with a Sandbox Apple ID. All features are accessible without purchasing in the Simulator.

Build & Archive in Xcode

Steps to Archive & Upload

  • Set scheme to HydroFlow, destination to Any iOS Device (arm64)
  • Bump version: CFBundleShortVersionString → e.g. 1.0.0
  • Bump build number: CFBundleVersion → e.g. 1
  • Xcode menu → Product → Archive
  • Organiser opens → click Distribute App
  • Select App Store Connect → Next
  • Select Upload → Next
  • Enable: Strip Swift symbols, Bitcode (if applicable), Upload symbols
  • Select Distribution Certificate & Provisioning Profile → Next → Upload
  • Wait for build to appear in App Store Connect (5–30 min)
💡 Run xcodegen generate before archiving to ensure project.pbxproj is up to date.
